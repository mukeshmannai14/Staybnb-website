# Staybnb Clone

This project is a responsive front-end clone of the Airbnb website built using **HTML5**, **CSS3**, and **JavaScript**. It recreates the look and feel of Airbnb's homepage with a clean, modern design and focuses on responsive layouts for different screen sizes.


## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Modern navigation bar
* Search section
* Property listing cards
* Interactive dropdown menu using JavaScript
* Clean and user-friendly interface
* Built using semantic HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Flexbox
* Media Queries

## 🔗 Live Demo

 **Website:** https://mukeshmannai14.github.io/Staybnb-website/


## 🎯 Purpose

This project was created to improve front-end development skills by practicing responsive web design, layout techniques, DOM manipulation, and interactive user interface development.

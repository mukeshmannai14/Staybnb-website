# Staybnb 

This project is a responsive front-end clone of the Airbnb website built using **HTML5** and **CSS3**. It recreates the look and feel of Airbnb's homepage with a clean, modern design and focuses on responsive layouts for different screen sizes.

## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Modern navigation bar
* Search section
* Property listing cards
* Clean and user-friendly interface
* Built using semantic HTML and CSS

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## 🎯 Purpose

This project was created to improve front-end development skills by practicing responsive web design, layout techniques, and UI implementation.
